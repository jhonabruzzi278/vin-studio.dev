## Descripción del Proyecto

Eduvin es una plataforma de formación online diseñada para transformar conocimiento en ingresos.
Centraliza la creación, venta y entrega de cursos digitales en una experiencia fluida para alumnos e instructores, permitiendo escalar una academia digital de forma profesional.

## El Desafío

Las empresas y marcas educativas suelen perder oportunidades por usar herramientas separadas para vender, enseñar y medir resultados.
Esto provoca procesos lentos, mala experiencia de usuario y poca visibilidad del desempeño del negocio.

## La Solución

Se construyó una solución integral que conecta todo el ciclo comercial y académico en un solo producto:

### Características Principales

- Catálogo de cursos atractivo y optimizado para conversión.
- Registro e inicio de sesión seguro para usuarios.
- Proceso de compra rápido con pagos online.
- Acceso inmediato al contenido tras la compra.
- Seguimiento de progreso y emisión de certificados.
- Panel de instructor con control de contenido y métricas clave.

## Resultados

- Mayor profesionalización de la oferta educativa digital.
- Mejor experiencia de compra y aprendizaje en una sola plataforma.
- Reducción de fricción operativa para equipos académicos/comerciales.
- Base tecnológica preparada para escalar cursos, usuarios y facturación.

## Tecnologías

- Next.js + React + TypeScript para una experiencia rápida y moderna.
- Tailwind CSS + Radix UI para una interfaz consistente y escalable.
- PostgreSQL + Prisma para gestión robusta de datos.
- Clerk para autenticación y gestión de usuarios.
- Stripe para cobros online y automatización de compras.
- UploadThing para gestión de archivos multimedia.
- Recharts para visualización de métricas de negocio.
